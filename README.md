# JAMII BOLT Frontend

Clean BOLT UI frontend for JAMII marketplace.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

This project is configured for Vercel deployment. Simply connect your GitHub repository to Vercel.

## Environment Variables

Set these in Vercel:
- `VITE_API_BASE_URL` - Your backend API URL (e.g., `https://jamii-backend-production.up.railway.app`)
